MiniUrl - URL Shortener Service.

This is as microservice created for the FreeCodeCamp URL Shortener micoservice challenge. Pass a URL in the format shown below, and you will get a json response with the shortened URL which will redirct you to the original URL. Enjoy!

Usage:

Input: https://sleepy-shelf-30869.herokuapp.com/new/https://google.com
Output: {"originalurl":"https://google.com","miniurl":"https://sleepy-shelf-30869.herokuapp.com/821406"}
